# react.js

transpiler babel: für JSX (Java Script Syntax Xtension)

## App erstellen

```bash
npx create-react-app reddit-live
```

## Dependencies installieren

```bash
npm i --save axios
```

## App starten

```bash
npm start
```

## Veröffentlichen via surge

```bash
npm run build
cd build
npx surge
```
